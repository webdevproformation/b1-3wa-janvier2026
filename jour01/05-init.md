# NOUS POUVONS CREER un REPOSITORY LOCAL 

si vous voulez créer une projet from scratch (à partir de zéro)

```sh
git init
```

va créer un dossier caché qui s'appelle `.git `
si vous voulez voir ce dossier, il faut saisir :

```txt
C:\Users\harri\Documents\formateur\git 3wa\documentation\260104 B1 Montparnasse\.git
```


si vous voulez récupérer un projet disponible sur github 

- git clone <addresse internet du dépôt>
